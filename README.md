## ⭐ &nbsp;Spotlight — NOVA E-Commerce Platform

> A production-ready full-stack e-commerce system built from scratch with a focus on **performance**, **security**, and **real-world reliability**.

<div align="center">

```
🛒 NOVA E-Commerce — Key Highlights
├── 🔐 Passwordless OTP authentication (no passwords stored)
├── ⚡ In-memory API caching with TTL + cache invalidation
├── 🔄 Real-time stock updates — prevents overselling race conditions
├── 💳 Stripe credit card + Cash on Delivery checkout
├── 📦 Live order tracking with 30s polling
├── 🛡️ Circuit breakers for Stripe, Cloudinary, Email services
├── 📱 Fully responsive — admin panel + storefront
├── 🚀 Gzip compression, lazy loading, code splitting
└── 📊 Core Web Vitals monitoring + performance alerting
```
